Fatoração

Resumo

Cada número inteiro pode ser escrito como a multiplicação de números primos(fatores). Para todo n inteiro existe uma única forma de escrevê-lo fatorado.

As Aplicações para a fatoração são inúmeras: desde encontrar maior divisor comum (MDC) a facilitar nos cálculos de raizes, portanto este é o primeiro algoritmo criado para este projeto, tendo em vista as possibilidades que ele abre.

Atenção!

Recomendamos fortemente que você não utilize esta ferramenta como forma de obter respostas de atividades. Use-a apenas como correção para exercícios ou em atividades não acadêmicas, já que resolver por si mesmo os exercícios é fundamental para a memorização do conteúdo.
